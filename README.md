# firebase.windows
Firebase for Windows (for example WPF) It will support Auth, Database and Storage.

現状でできることは以下のとおりです

* WPFからFirebaseを利用
* Email-Password、Twitter、Facebook、GitHub、匿名認証、サインアウト

これから作りたいなと思ってるのは以下のとおりです

* 認証まわりを強化
* Database
* Storage
